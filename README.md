## ✨ Features

* **📦 Unified Mod Browser:** Search, filter, and install mods, resource packs, shaders, and data packs directly from **Modrinth** and **CurseForge** without leaving the app.
* **🛡 Intelligent Compatibility Filter:** Automatically hides incompatible mod versions by matching them to your current Minecraft installation and mod loader.
* **☕ Zero-Configuration Java Manager:** No manual Java installation required. Onyx automatically downloads and configures the optimal JDK version matching your game architecture.
* **⚡ Multi-Loader Support:** Instantly install Fabric, Forge, NeoForge, and Quilt mod loaders with a single click.
* **🎮 Discord Rich Presence:** Share your gaming status (active instance, playtime, and version) directly to your Discord profile.
* **👤 Custom Skin Support:** Change and manage skins easily, even for offline accounts.
* **🎨 Fully Native Design:** A premium, lightweight macOS interface featuring smooth micro-animations, inline settings, and customized theme support.
* **🔄 Seamless Updates:** Built-in automatic updates ensure you always have the latest improvements and bug fixes.

---

## 🛠 System Requirements

* **Operating System:** macOS 14.0 (Sonoma) or newer.
* **Hardware Support:** Native support for both Apple Silicon (M1/M2/M3/M4) and Intel-based Macs.

---

## 📦 Installation

1. Navigate to the [Releases](https://github.com/deeppacketdev-hub/onyx-launcher/releases) page.
2. Download the latest `.zip` or `.pkg` installation bundle.
3. Drag the app into your `Applications` folder or run the installer package.

> [!TIP]
> Since the application is signed with an ad-hoc local signature, macOS Gatekeeper might block its execution. To open it, simply right-click the app in Finder and choose **Open**.
